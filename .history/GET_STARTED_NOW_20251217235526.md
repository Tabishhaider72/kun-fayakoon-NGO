## üî• TROUBLESHOOTING

### Problem: CORS Error in Console
**Solution:**
1. Refresh page: `Ctrl+Shift+R`
2. Check `.env.local` has all values
3. Restart server: Stop with Ctrl+C, run `pnpm dev` again

### Problem: Upload Button Disabled
**Solution:**
1. Clear browser cache: `Ctrl+Shift+Delete`
2. Refresh page: `Ctrl+Shift+R`
3. Try again

### Problem: File Upload Fails
**Reasons:**
- File too large (> 500 KB) ‚Üê Reduce size
- Wrong format (not image) ‚Üê Use JPEG/PNG/WebP/GIF
- Image too small (< 400px width) ‚Üê Use larger image

### Problem: Login Fails
**Solution:**
1. Password is: `admin123` (case-sensitive)
2. Or check `.env.local` for custom password
3. Refresh page and try again

### Problem: Image Not Appearing
**Debug:**
1. Go to `console.firebase.google.com`
2. Select "kun-fayakoon" project
3. Click "Firestore Database"
4. Look for "gallery" collection
5. Should have your document there
6. If not, check browser console (F12) for errors

### Problem: "Connection failed" Error
**Solution:**
1. Check internet connection
2. Check Firebase project is active
3. Check `.env.local` has correct credentials
4. Refresh page and try again

**Need more help?** ‚Üí Read `TROUBLESHOOTING.md`
